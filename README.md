# Analog Clock

A simple and responsive **Analog Clock** built using  
**HTML, CSS and JavaScript**.

This project shows the current system time using rotating clock hands just like a real wall clock.

---

## Features

- Real-time working clock
- Hour, Minute & Second hands
- Smooth rotation
- Fully responsive design
- Clean & modern UI
- Works on all browsers
